MODE D'EMPLOI ULTRA SIMPLE

1. Crée un compte GitHub si besoin.
2. Crée un nouveau dépôt public.
3. Téléverse tout le contenu de ce dossier dans le dépôt.
4. Sur GitHub : Settings > Pages > Source = GitHub Actions.
5. Attends 1 à 2 minutes.
6. Ton lien sera : https://TON-USERNAME.github.io/NOM-DU-DEPOT/

Si tu préfères, tu peux aussi glisser-déposer ces fichiers dans GitHub via l'interface web.
